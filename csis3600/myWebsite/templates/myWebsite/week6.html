1. Describe in detail how the File Manager allocates a file to a single user.

The file manager resolves the physical address of the file and passes theis to the Device Manager. The device manager turns on the sppropriate device driver and this activates reading a portion of the file into an I/O buffer. The device manager then alerts the memory manger that the I/O buffer is ready to be loaded into RAM. The memory manager allocated memory for the data sitting in th eI/O buffer and communicates when it is finished to the File Manager.

List the Linux commands we convered today and what they do.

  • cd

    Change Directory

  • pwd

    See the present working directory

  • ls -al # Or any other variation

    Gives the long listing of the designated directory

  • echo "hello world!" > hello.txt

    Redirects the output "Hello World!" to the file hello.txt

  • echo "Zebra" >> hello.txt

    Appends the output to hello.txt

  • cat hello.txt

    Prints the contents of hello.txt

  • cat /etc/passwd

    Prints the users on the machine

  • rm ~/hello.txt

    Removes hello.txt

  • man ls

    Look at the manual pages for ls

  • cat /etc/passwd | less

    print /etc/passwd and send the output to less

  • tail /etc/passwd

    Prints the last lines of /etc/passwd

  • history

    Prints out the history

  • alias ll='ls -al'

    Sets the specified alias

  • source ~/.bashrc

    Reloads the bash shell
